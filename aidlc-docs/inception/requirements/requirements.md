# Análise de Requisitos — Análise de Mercado Competitiva

## Intent Analysis

- **Solicitação do Usuário**: Implementar análise de mercado validando informações com os sites dos concorrentes (TOTVS e LG lugar de gente)
- **Tipo de Requisição**: Análise de Mercado / Pesquisa Competitiva
- **Escopo**: Mercado de software empresarial brasileiro (ERP + HCM/RH)
- **Complexidade**: Moderada — requer pesquisa web, síntese de dados e análise estruturada
- **Concorrentes Analisados**:
  - TOTVS: https://www.totvs.com/
  - LG lugar de gente: https://www.lg.com.br/

## Requisitos Funcionais

1. Verificar se a solução proposta já existe no mercado (análise de existência)
2. Avaliar aderência competitiva em relação aos concorrentes identificados
3. Calcular e apresentar TAM, SAM e SOM do mercado-alvo
4. Validar informações cruzando dados dos sites dos concorrentes com fontes de mercado

## Requisitos Não-Funcionais

1. Dados baseados em fontes verificáveis (sites oficiais + relatórios de mercado)
2. Análise em português brasileiro
3. Estrutura clara e navegável em markdown
4. Referências e fontes citadas para rastreabilidade
